EHS Field Report

Simple field reporting for near misses, hazardous conditions, safety recognition, and stop controls.

This page is built to work as a GitHub Pages site. Once you add your live form endpoint, submissions can go straight to your email. A second webhook can also send instant phone alerts.

Setup notes: replace YOUR_FORM_ID with your Formspree form ID. If you want text alerts, paste your webhook URL into the smsWebhook value. Then publish this file as index.html in a new GitHub repo and turn on GitHub Pages.
